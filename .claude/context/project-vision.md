# Project Vision

## Long-Term Vision

成為台灣學生申請美國大學的**首選 AI 助手**，從選校到錄取全程陪伴。

## Mission Statement

透過 AI 技術降低美國升學資訊門檻，讓每位學生都能獲得專業級的申請指導。

## Strategic Goals

### Short-Term (MVP)
- 建立可運作的選校推薦系統
- 驗證 AI 推薦準確性
- 建立顧問轉化漏斗
- 獲得首批用戶回饋

### Mid-Term (Phase 2-3)
- 擴展功能：Essay 輔助、時間軸規劃
- 擴大學校資料庫（Top 300）
- 建立用戶帳號系統
- 優化推薦演算法

### Long-Term
- 全方位升學顧問 AI
- 涵蓋申請全流程
- 國際學生社群
- 多國家支援

## Product Roadmap

```
MVP (現在)
├── 選校推薦

```

## Strategic Priorities

1. **用戶體驗優先** - 簡單、快速、準確
2. **資料品質** - 推薦準確是核心競爭力
3. **轉化效率** - 有效引導至顧問服務
4. **可擴展性** - 架構支援未來功能

## Success Indicators

### MVP 成功標準
- 30 天 100+ 顧問預約
- 推薦準確率 > 80%
- 用戶滿意度 > 4.0/5.0

### 長期成功標準
- 成為台灣升學顧問市場知名品牌
- 穩定的顧問轉化管道
- 用戶口碑推薦

## Competitive Advantage

1. **免費入門** - 降低使用門檻
2. **即時回應** - 不需等待顧問時間
3. **AI 一致性** - 穩定的推薦品質
4. **與真人結合** - AI + 顧問的最佳組合
