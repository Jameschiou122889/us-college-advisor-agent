# Product Context

## Product Definition

**US College Advisor Agent** 是一個 AI 驅動的美國升學選校顧問，專為台灣學生和家長設計。

## Target Users

### Primary: 高中生 (Undergraduate 申請者)
- **特徵**: 正在準備或考慮申請美國大學
- **痛點**: 不知道自己的條件適合哪些學校
- **目標**: 快速獲得個人化選校建議
- **使用情境**: 課後/週末，獨立或與家長一起使用

### Secondary: 大學生 (Graduate 申請者) 【Phase 2 新增】
- **特徵**: 準備申請美國研究所 (Master/PhD)
- **痛點**: 研究所選校考量因素不同
- **目標**: 了解適合的研究所和錄取機會
- **使用情境**: 大三/大四，規劃留學

### Tertiary: 家長
- **特徵**: 子女準備申請美國大學的家長
- **痛點**: 對美國大學系統不熟悉，擔心選錯學校
- **目標**: 了解可行選項和預算範圍
- **使用情境**: 替子女初步研究，或與子女一同使用

## User Personas

### Persona 1: 小明 (17歲，高二生)
```
背景: GPA 3.7，正在準備 SAT，對 CS 有興趣
動機: 想知道自己能申請哪些學校
行為: 習慣用手機查資料，喜歡快速得到答案
期望: 5 分鐘內獲得推薦清單
```

### Persona 2: 王媽媽 (45歲，家長)
```
背景: 女兒高一，開始規劃升學
動機: 了解美國大學選項和費用
行為: 會仔細研究，注重性價比
期望: 清楚的學費資訊和專業顧問諮詢
```

## Core Features (MVP) ✅

### 1. 學生資料輸入
- 彈性表單：必填 + 選填欄位
- 支援「尚未考試」選項
- Email 欄位收集

### 2. AI 選校推薦
- 分層推薦：衝刺校/適中校/安全校
- 顯示錄取機率估算
- 考慮科系競爭度

### 3. 結果展示
- 10 所推薦學校 (Reach 3 / Match 3 / Safety 4)
- 每校顯示排名、學費、熱門科系
- 響應式設計
- **Email 寄送推薦報告** ✅

### 4. 顧問轉化
- 結果頁 CTA
- 預約表單 (自動帶入 Email)
- 聯絡資訊收集
- **Airtable 儲存** ✅

## Phase 2 Features (規劃中)

### 5. 成績單上傳
- 多檔案上傳支援 (PDF/JPG/PNG)
- OCR 自動解析
- 自動填入 GPA/SAT/課程

### 6. 進階欄位
- 申請層級 (Undergraduate/Graduate)
- GPA 多制度轉換 (4.0/4.3/5.0/百分制)
- 修課紀錄 (AP/IB/榮譽課程)

### 7. 多語言支援
- 中文/英文切換
- AI 分析結果多語言輸出

## Use Cases

### UC-1: 快速選校評估
```
前提: 學生有基本成績資料
流程:
1. 填寫 GPA 和目標科系
2. 選填 SAT/TOEFL 分數
3. 提交獲得推薦
結果: 獲得 15 所分層推薦學校
```

### UC-2: 預算導向選校
```
前提: 家長關心學費
流程:
1. 設定學費預算範圍
2. 填寫學生基本資料
3. 查看符合預算的學校
結果: 推薦學校皆在預算內
```

### UC-3: 顧問預約
```
前提: 用戶查看推薦結果後
流程:
1. 點擊「預約顧問」CTA
2. 填寫聯絡資訊
3. 選擇期望諮詢時間
結果: 顧問收到預約通知
```
