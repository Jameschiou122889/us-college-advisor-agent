# Task: 設計推薦 Prompt

## Description

設計 Claude API 的 prompt template，用於根據學生條件分析並推薦適合的美國大學，輸出分層推薦（衝刺/適中/安全校）。

## Acceptance Criteria

- [ ] 建立 `prompts/recommendation.md` prompt template
- [ ] Prompt 能正確解析學生輸入資料
- [ ] 輸出格式為結構化 JSON
- [ ] 包含 Few-shot 範例提升準確度
- [ ] 推薦邏輯包含 GPA、SAT、科系競爭度因素

## Technical Details

### Prompt 結構
```markdown
# 選校推薦系統

## 角色
你是專業的美國升學顧問...

## 學生資料
{{student_data}}

## 學校資料庫
{{schools_data}}

## 任務
根據學生條件，推薦學校並分類...

## 輸出格式
{
  "reach": [...],
  "match": [...],
  "safety": [...]
}

## 範例
[Few-shot examples]
```

### 推薦邏輯
- Reach: 錄取機率 < 30%
- Match: 錄取機率 30-70%
- Safety: 錄取機率 > 70%
